# Cyber Lab Toolkit

A Python-based toolkit to automate lab-based reconnaissance and exploitation using Nmap, Hydra, Wireshark (tshark), and Metasploit.

> ⚠️ Use this **only in a controlled lab environment**.

## Features

- Nmap Port Scanning
- Hydra FTP Brute Force
- Tshark Packet Capture
- Metasploit RPC Exploit

## Installation

```bash
git clone https://github.com/yourusername/cyber-lab-toolkit.git
cd cyber-lab-toolkit
pip install -r requirements.txt
